Setup guide — read this first
1
Create a Notion account
Go to notion.so and sign up for free. It's like a smart notebook.
2
Create a new database
In Notion, click + New page → choose Table. Name it Tony Tasks. Add columns: Name (title), Category (select), Date (date), Done (checkbox), Notes (text).
3
Get your Notion API key
Go to notion.so/my-integrations → click New integration → name it TonyOS → copy the key that starts with secret_
4
Connect the database to your integration
Open your Tony Tasks database in Notion → click the ... menu top right → Add connections → find TonyOS and connect it.
5
Get your Database ID
Open Tony Tasks in your browser. The URL looks like: notion.so/abc123.../YOUR-ID-HERE?v=.... Copy the 32-character ID between the last / and the ?.
6
Paste both above and save
Put the secret_ key and the database ID in the fields above. Done. Every task you log here will appear in your Notion automatically.
7
Add to your phone home screen
In Safari (iPhone) or Chrome (Android): tap the share button → Add to Home Screen. It will look like an app. One tap access, always.
